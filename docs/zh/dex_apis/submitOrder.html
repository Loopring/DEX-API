下订单

API 摘要

功能

下买单或卖单。

HTTP 方法

POST

访问路径

/api/v2/order

API 描述

请求参数

字段 类型 是否必现 说明 举例
exchangeId integer 是 交易所ID 1
orderId integer 是 订单ID 1
accountId integer 是 账户ID 1
tokenSId integer 是 selling token id in exchange 0
tokenBId integer 是 buying token id in exchange 2
amountS string 是 selling amount of token, decimal string in WEI 1000000000000000000
amountB string 是 buying amount of token, decimal string in WEI 1000000000000000000
allOrNone boolean 否 all or none false
buy boolean 否 buy true
validSince integer 是 beginning period of this order, timestamp in second 1567053142
validUntil integer 是 ending period of this order, timestamp in second 1567053142
maxFeeBips integer 是 max accepted fee of this order, selling amount times x/10000 20
label integer 是 label 20
signatureRx string 是 trading signature Rx of this order, decimal string 13375450901292179417
15497484957179306991
15173547203971250276
33242680470075859
signatureRy string 是 trading signature Ry of this order, decimal string 13375450901292179417
15497484957179306991
15173547203971250276
33242680470075859
signatureS string 是 trading signature S of this order, decimal string 13375450901292179417
15497484957179306991
15173547203971250276
33242680470075859
clientOrderId string 否 client specified ID of this order 1
channelId string 否 client specified channel ID of this order 1

    "tokenBId" : 2,

    "amountB" : "1000000000000000000",
    "allOrNone" : false,
    "buy" : true,
    "validSince" : 1567053142,
    "validUntil" : 1567053142,
    "maxFeeBips" : 20,
    "label" : 20,
    "signatureRx" : "13375450901292179417<br/>15497484957179306991<br/>15173547203971250276<br/>33242680470075859",

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 返回结果 /

响应示例

{
    "resultInfo" : {
        "code" : 0,
        "message" : "SUCCESS"
    }
}

公共信息

访问地址

http://localhost:9000

请求头

字段 类型 是否必现 说明 举例
X-API-KEY string 否 API Key. 写操作接口需要 "sra1aavfa"
X-API-SIG string 否 签名信息. getapikey, cancelorder 接口需要 "dkkfinfasdf"

响应

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 调用结果 /

返回码

返回码 描述
0 成功
100000 内部未知错误
100001 参数非法

模型

ResultInfo

字段 类型 是否必现 说明 举例
code number 是 返回码 0
message string 是 返回消息. 该返回消息更多用来调试，不要直接在前端显示 "SUCCESS"
