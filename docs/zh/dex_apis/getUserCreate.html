获取用户开户记录

获取用户开户记录

API 摘要

HTTP 方法

GET

访问路径

/api/v2/user/createInfo

功能

获取用户开户记录

API 描述

请求参数

字段 类型 是否必现 说明 举例
accountId integer 是 用户ID 1
start integer 否 起始时间 1578558098000
end integer 否 结束时间 1578558098000
status string 否 充值状态 processing
limit integer 否 限制条数 50
fromHash string 否 锚地HASH(会返回该条hash后面的数据，用于翻页) 0x9d114267e8b261457d
567093c13cf3deea5f14
c9235be26c6fa833dba1
2a9632
offset integer 否 偏移量 1

请求示例

curl https://api.loopring.io/api/v2/user/createInfo?accountId=1&start=1578558098000&end=1578558098000&status=processing&limit=50&fromHash=0x9d114267e8b261457d567093c13cf3deea5f14c9235be26c6fa833dba12a9632&offset=1

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 API返回信息 /
data TxDataList 否 用户操作记录列表对象 /

返回码

返回码 描述
100000 ERR_INTERNAL_UNKNOWN
107001 ERR_USERTX_ACCOUNT_ID_INVALID
104001 ERR_REST_EXPECT_APIKEY
104002 ERR_REST_APIKEY_NOT_MATCH
104003 ERR_REST_ACCOUNT_NOT_EXIST

模型

ResultInfo

API返回信息

字段 类型 是否必现 说明 举例
code integer 是 返回码 0
message string 是 返回信息 SUCCESS

TxDataList

用户操作记录列表对象

字段 类型 是否必现 说明 举例
totalNum integer 是 操作记录符合条件的总条数，用于分页 1
transactions list< TxData > 是 操作记录当前页列表 /

TxData

用户操作记录列表对象

字段 类型 是否必现 说明 举例
id integer 是 操作记录唯一ID 1
txType string 是 操作类型 deposit
symbol string 是 操作币种 LRC
amount string 是 申请数量 1000000000000000000
realAmount string 是 实际数量 1000000000000000000
receiver integer 否 接收方 1
txHash string 是 交易HASH 0x9d114267e8b261457d
567093c13cf3deea5f14
c9235be26c6fa833dba1
2a9632
feeAmount string 是 操作类型 1000000000000000
status string 是 当前操作的状态 processing
progress string 是 进度 100%
timestamp integer 是 更新时间 1578572292000
depositType string 否 充值类型 DEPOSIT
blockNum integer 否 块高度 100
