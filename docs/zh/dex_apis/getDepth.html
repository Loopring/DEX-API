获取市场深度信息

获取市场深度信息

API 摘要

HTTP 方法

GET

访问路径

/api/v2/depth

功能

获取某个市场对的深度信息

API 描述

请求参数

字段 类型 是否必现 说明 举例
market string 是 市场对 LRC-ETH
level integer 是 深度等级，越大表示合并的深度越多 2
limit integer 否 返回条数限制 2

请求示例

curl https://api.loopring.io/api/v2/depth?market=LRC-ETH&level=2&limit=2

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 API返回信息 /
data Depth 否 深度信息 /

响应示例

{
    "resultInfo" : {
        "code" : 0,
        "message" : "SUCCESS"
    },
    "data" : {
        "version" : 147,
        "timestamp" : 432312312,
        "bids" : [
            {
                "price" : "0.002",
                "size" : "21000",
                "volume" : "33220000",
                "count" : 4
            }
        ],
        "asks" : [
            {
                "price" : "0.002",
                "size" : "21000",
                "volume" : "33220000",
                "count" : 4
            }
        ]
    }
}

返回码

返回码 描述
100000 ERR_INTERNAL_UNKNOWN
108000 ERR_DEPTH_UNSUPPORTED_MARKET
108001 ERR_DEPTH_UNSUPPORTED_LEVEL

模型

ResultInfo

API返回信息

字段 类型 是否必现 说明 举例
code integer 是 返回码 0
message string 是 返回信息 SUCCESS

Depth

深度信息

字段 类型 是否必现 说明 举例
version integer 是 版本号 147
timestamp integer 是 时间戳 432312312
bids list< Slot > 是 买单深度 /
asks list< Slot > 是 卖单深度 /

Slot

每一条深度数据

字段 类型 是否必现 说明 举例
price string 是 价格 0.002
size string 是 挂单量 21000
volume string 是 挂单总量 33220000
count integer 是 聚合的订单数目 4
