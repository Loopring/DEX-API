获取市场深度信息

获取市场深度信息

API 摘要

功能

获取某个市场对的深度信息。

HTTP 方法

GET

访问路径

/api/v2/depth

API 描述

请求参数

字段 类型 是否必现 说明 举例
market string 是 市场对 LRC-ETH
level integer 是 深度等级，越大表示合并的深度越多 2
limit integer 否 返回条数限制 2

请求示例

curl http://localhost:9000/api/v2/depth?market=LRC-ETH&level=2&limit=2

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 返回结果 /
depth Depth 否 深度信息 /

响应示例

{
    "resultInfo" : {
        "code" : 0,
        "message" : "SUCCESS"
    },
    "depth" : {
        "version" : 147,
        "timestamp" : 432312312,
        "bids" : [
            {
                "price" : "0.002",
                "size" : "21000",
                "volume" : "33220000",
                "count" : 4
            }
        ],
        "asks" : [
            integer
        ]
    }
}

返回码

返回码 描述
108000 ERR_DEPTH_UNSUPPORTED_MARKET
108001 ERR_DEPTH_UNSUPPORTED_LEVEL

模型

Depth

深度信息

字段 类型 是否必现 说明 举例
version integer 是 版本号信息 147
timestamp integer 是 时间戳 432312312
bids list< Slot > 是 卖单深度 /
asks list< integer > 是 买单深度 /

Slot

每一条深度数据

字段 类型 是否必现 说明 举例
price string 是 价格 0.002
size string 是 买方成交量 21000
volume string 是 卖方成交量 33220000
count integer 是 聚合的订单数目 4

公共信息

访问地址

http://localhost:9000

请求头

字段 类型 是否必现 说明 举例
X-API-KEY string 否 API Key. 写操作接口需要 "sra1aavfa"
X-API-SIG string 否 签名信息. getapikey, cancelorder 接口需要 "dkkfinfasdf"

响应

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 调用结果 /

返回码

返回码 描述
0 成功
100000 内部未知错误
100001 参数非法

模型

ResultInfo

字段 类型 是否必现 说明 举例
code number 是 返回码 0
message string 是 返回消息. 该返回消息更多用来调试，不要直接在前端显示 "SUCCESS"
