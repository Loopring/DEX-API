获取订单详情

获取订单详情

API 摘要

HTTP 方法

GET

访问路径

/api/v2/order

功能

根据指定hash获取订单详情

API 描述

请求参数

字段 类型 是否必现 说明 举例
accountId integer 是 用户ID 1
orderHash string 是 订单HASH 13375450901292179417
15497484957179306991
15173547203971250276
33242680470075859

请求示例

curl https://api.loopring.io/api/v2/order?accountId=1&orderHash=13375450901292179417154974849571793069911517354720397125027633242680470075859

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 API返回信息 /
data OrderDetail 否 订单详情 /

响应示例

{
    "resultInfo" : {
        "code" : 0,
        "message" : "SUCCESS"

    "data" : {

        "size" : "8210000000000001024",

100000 ERR_INTERNAL_UNKNOWN
107003 ERR_USERTX_ORDER_HASH_EMPTY
107004 ERR_USERTX_ORDER_NOT_EXIST
104001 ERR_REST_EXPECT_APIKEY
104002 ERR_REST_APIKEY_NOT_MATCH
104003 ERR_REST_ACCOUNT_NOT_EXIST

模型

ResultInfo

API返回信息

字段 类型 是否必现 说明 举例
code integer 是 返回码 0
message string 是 返回信息 SUCCESS

OrderDetail

订单详情信息

字段 类型 是否必现 说明 举例
hash string 是 订单HASH 0xfb5e711c2f044e9432
2ed262229cd8f0d0da00
c22e1a00a0f5d881e45a
38e1cf
clientOrderId string 是 客户端订单唯一ID 200310143135081332
size string 是 买方成交总量 8210000000000001024
volume string 是 卖方成交总量 163182534700000032
price string 是 成交价格 0.01987608
filledSize string 是 买方已成交量 0
filledVolume string 是 卖方已成交量 0
filledFee string 是 已生成手续费 0
status string 是 订单状态 processing
validSince integer 是 生效起始时间 1583929591
validUntil integer 是 生效截止时间 1586521591
createdAt integer 是 订单创建时间 1583933191494
side string 是 买卖方向 SELL
market string 是 市场对 LRC-ETH
