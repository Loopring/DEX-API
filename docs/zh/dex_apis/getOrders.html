根据用户订单列表

根据用户订单列表

API 摘要

HTTP 方法

GET

访问路径

/api/v2/orders

功能

获取指定用户的订单列表，可以根据市场对、起始时间等条件过滤

API 描述

请求参数

字段 类型 是否必现 说明 举例
accountId integer 是 用户ID 1
market string 否 市场对 LRC-ETH
start integer 否 起始时间,单位ms 1567053142000
end integer 否 结束时间,单位ms 1567053242000
fromHash string 否 锚定hash（会返回该hash以后的数据） 26001051253364689664
17510367500403435128
94150245200567415610
3328855968837178
side string 否 买卖方向 buy
status string 否 订单状态列表 processing,processed
limit integer 否 限定数据条数（默认50条） 50
offset integer 否 偏移量（默认0） 0

请求示例

curl https://api.loopring.io/api/v2/orders?accountId=1&market=LRC-ETH&start=1567053142000&end=1567053242000&fromHash=2600105125336468966417510367500403435128941502452005674156103328855968837178&side=buy&status=processing,processed&limit=50&offset=0

响应字段

字段 类型 是否必现 说明 举例
resultInfo ResultInfo 是 API返回信息 /
data OrdersDetail 否 订单列表返回结果 /

模型

API返回信息

字段 类型 是否必现 说明 举例

订单列表返回结果

字段 类型 是否必现 说明 举例

orders list< OrderDetail > 否 订单列表 /
