Get user's next Ethereum nonce

Get user's next Ethereum nonce

Rate limit: 5 every second

API Overview

HTTP method

GET

Path

/api/v2/ethNonce

Summary

Returns the next nonce of the Ethereum address associated with the user's account

API description

ZLnN"

Field Type Required Description Example

00"

HTTP
