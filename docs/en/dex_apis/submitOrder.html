下订单

API Overview

Function

下买单或卖单。

HTTP method

POST

Visit path

/api/v2/order

API description

Request parameters

Field Type Required Description Sample
exchangeId integer Y 交易所ID 1
orderId integer Y 订单ID 1
accountId integer Y 账户ID 1
tokenSId integer Y selling token id in exchange 0
tokenBId integer Y buying token id in exchange 2
amountS string Y selling amount of token, decimal string in WEI 1000000000000000000
amountB string Y buying amount of token, decimal string in WEI 1000000000000000000
allOrNone boolean N all or none false
buy boolean N buy true
validSince integer Y beginning period of this order, timestamp in second 1567053142
validUntil integer Y ending period of this order, timestamp in second 1567053142
maxFeeBips integer Y max accepted fee of this order, selling amount times x/10000 20

Common info

API host

http://localhost:9000

Header

Field Type Required Description Sample
X-API-KEY string N API Key. Needed by updating api "sra1aavfa"
X-API-SIG string N Signature info. Needed by getapikey and cancelorder api "dkkfinfasdf"

Response

Field Type Required Description Sample
resultInfo ResultInfo Y Result of api invoking /

Return code

Return code Description
0 Success
100000 Internal unknown error
100001 Illegality parameter

模型

ResultInfo

Field Type Required Description Sample
code number Y Return code 0
message string Y Return message. This is used for debug only. Do not show to users "SUCCESS"
