#### ResultInfo
<span id="ResultInfo"></span>

| 字段 |  类型 | 必须 | 说明 | 举例 |
| :---- | :---- | :---- |   :----   |  :--- |
| code | integer | 是 | 返回码 | 0 |

| 100202 | 更新失败 |
| 100203 | 内部存储错误 |
| 100204 | 重复提交 |
